# 🏡 Airbnb Clone (Node.js + Express + MongoDB)

A backend-focused Airbnb-style application built using Node.js, Express, and MongoDB.  
This project demonstrates REST API design, authentication, MVC architecture, and database integration.

---

## 🚀 Features

- 👤 User Authentication (Signup / Login / Logout)
- 🏠 Add, Edit, Delete Home Listings (Host functionality)
- 🔍 Explore Available Homes
- ❤️ Add to Favorites (if implemented)
- 📦 RESTful API structure
- 🗄️ MongoDB with Mongoose ODM
- 🧩 MVC Architecture (Model - View - Controller)
- 🔐 Session-based Authentication (or JWT if used)

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- Express Session
- Multer (File Uploads)
- EJS (if used)
- Git & GitHub
